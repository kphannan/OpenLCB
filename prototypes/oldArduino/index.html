OpenLCB Arduino Prototype

This is the introductory page for the OpenLCB Arduino prototypes code.

Organization

There are two main subdirectories:

  • sketches – This contains Arduino “sketches” to download into boards. There are different sketches for different purposes:

    • OlcbBasicNode – basic producer/consumer board with 2 pins of input and 2 pins of output

    • OlcbServoNode – drive a servo from OpenLCB events

  • libraries – This contains the main “OpenLCB” library, along with the “CAN” library needed for access to the CAN controller hardware.

Getting the Code

The content of this development area, including code, documents and web pages, is kept in the OpenLCB SourceForge.net SVN repository in the “trunk/prototypes/Arduino” directory. To check it out, please do:

svn co http://svn.code.sf.net/p/openlcb/svn/trunk/prototypes/Arduino Arduino

For more information on the SVN source code management tool, please see the SVN website and the online SVN book.

After downloading the code, you need to make the libraries available to the Arduino IDE. How to do this depends on the Arduino version and computer type you're using. On Mac and Linux, make a symolic link from your "Arduino/libraries" directory to the individual directories in the "libraries" directory you just checked out. On Windows, copy the directories into the right location.
