OpenLCB AVR-C Prototype

This is the index page of the OpenLCB prototype for AVR hardware.


See also the Ardunio prototype, which uses similar hardware and has a simpler IDE.

Getting the Code

The content of this development area, including code, documents and web pages, is kept in the OpenLCB SourceForge.net SVN repository in the “trunk/prototypes/avr-c” directory. To check it out, please do:

svn co http://svn.code.sf.net/p/openlcb/svn/trunk/prototypes/avr-c avr-c

For more information on the SVN source code management tool, please see the SVN website and the online SVN book.
