OpenLCB iOS Prototype Tests Area

This page describes some test infrastructure for the iOS applications.
  • run-mdns - Advertise a MDNS port that will provide data
  • send-data - Serve sample data to any connecting programs
  • data.txt - File of sample data
  • first - a shell script to send one line at a time from a file
These were developed on Mac OS X, but should work on any networking-aware OS.
