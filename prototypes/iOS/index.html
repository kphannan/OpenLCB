OpenLCB iOS Prototypes Area

We are creating iOS (iPhone) prototypes for OpenLCB. These talk via network connections, typically wireless.

Released versions of these will be available for free via the Apple App Store. You can also build them from OpenLCB SVN using XCode, and then install them directly to your phone.

Monitor

“Monitor” provides a traffic display for a CAN-based OpenLCB layout.

OlcbTools

“OlcbTools” is a multi-pane application that lets you browse and configure the OpenLCB nodes on your layout.
