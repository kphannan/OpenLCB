OpenLCB iOS Prototypes Area

We are creating iOS (iPhone) prototypes for OpenLCB. These talk via network connections, typically wireless.

Released versions of these will be (are) available via the Apple App Store. You can also build them from OpenLCB SVN using XCode, and then install them directly to your phone.
