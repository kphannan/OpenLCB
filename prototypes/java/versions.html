OpenLCB Java Prototype Versions

This describes the numbered versions of the prototype implementation(s) in Java of the proposed OpenLCB™ specification.

Versions are documented within the distributed Jar files using the manifest file. The version number is currently not available via a programmatic interface.

The SVN development code is generally done with the next number in the release sequence. The number should be in interpreted as major.minor, not a decimal number. 0.9 would be followed by 0.10.

    1. – first packaged Jar version

    2. - currently under development
