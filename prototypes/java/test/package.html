(This doesn't appear in the JavaDoc, as it's not in a package directory; it's more for people browsing the code)

Provides test code, as opposed to sample implementations, for OpenLCB.

Subpackages:

  • org.openlcb - unit tests of specific classes in the src tree
  • tools - tool and simulation implementations
  • scenarios - test the code by invoking specific scenarios
  • simulations - Monte-Carlo and event-based simulations of various OpenLCB operations; not tests of operational code
