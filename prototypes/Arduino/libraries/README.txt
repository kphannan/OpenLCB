These directory contains Arduino libraries.

The easiest way to use them is to put a symbolic link to each library directory into your Arduino IDE "libraries" directory, e.g. APP/hardware/libraries/

Contents:

NMRAnet: initial tools, mostly for CAN-based implementations

Ethernet2: Modified Ethernet library for Arduinos with pin 9 wired to 
Ethernet SPI CS, so that CAN can co-exist.

CAN: Basic CAN hardware support for LEDuino

LocoNet:  LocoNet communications
