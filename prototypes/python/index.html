OpenLCB Python Prototype

This is the index page of the OpenLCB Python prototypes directory.

Initial content for this is being developed, stay tuned.

Utilities and Modules

The basic Ethernet connection is accessed via the "ethernetolcblink" module. 

    import ethernetolcblink as link
    link.sendframe('10.0.1.98', 23, ':X180A000FN;', True)

Other cross-connection utilities can found in the "canolcbutils" module.

Tools and Tests

ethernetolcblink
Allows you to send specific packets
verifyNodeGlobal
Send a "Verify Node Global" frame, getting a reply from each connected node.
To run one from the Mac OS X or Linux command line, do e.g.: 
./verifyNodeGlobal.py
