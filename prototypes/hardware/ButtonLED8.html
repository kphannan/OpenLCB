ButtonLED8

ButtonLED8 is a small board designed to be used in prototyping and experimentation. It is designed to be directly compatible with the DevKit's Io nodes, connecting with their Input and Output connectors, or to their 2x5 auxillary connectors.

Each ButtonLED8 board has eight buttons and eight LEDs. Each pair connects to one i/o pin, and can operate independantly with the correct software. There is a ButtonLED library which implements flashing patterns on the LEDs and reads the button state.
