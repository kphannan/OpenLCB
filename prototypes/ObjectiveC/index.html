OpenLCB Objective-C Prototype Code

We are creating OpenLCB™ Objective-C libraries for use in iOS (iPhone) and OS X applications.

These are written in standard Objective-C to make them portable. To allow use of platform-specific code, there are three product libraries:

OpenLcbLib

General Objective-C code, without refererence to library/framework code not present on both iOS and OS X

OpenLcbiOSLib

Uses iOS-specific libraries and frameworks

OpenLcbOsXLib

Uses OS X-specific libraries and frameworks
