OpenLCB XML Prototype

This is the index page of the OpenLCB™ XML prototypes directory. This directory contains an OpenLCB prototype implementation for the XML component(s) of OpenLCB.

The initial content is a sample XML document for Configuration Description Information (CDI), along with an XML Schema for CDI and a XSLT stylesheet for displaying CDI documents as HTML.

Getting the Code

The content of this development area, including code, documents and web pages, is kept in the OpenLCB SourceForge.net SVN repository in the “trunk/prototypes/C” directory. To check it out, please do:

svn co http://svn.code.sf.net/p/openlcb/svn/trunk/prototypes/xml xml

For more information on the SVN source code management tool, please see the SVN website and the online SVN book.
