S9.6 Development Prototypes Area

This is the index page of the S9.6 development prototypes area.

File Tree Organization

This area contains a number of subdirectories organized by language, As a special case, there's also an “Arduino” subdirectory for Arduino sketches and libraries.

Aduino

C

java

Each in turn can have multiple projects and libraries in subdirectories.

Access

The content of this development area, including code, documents and web pages, is kept in the NMRAnet SourceForge.net SVN repository in the “s96/trunk/prototypes” directory. To check it out, please do:

svn co https://nmranet.svn.sourceforge.net/svnroot/nmranet/s96/trunk/prototypes s96

For more information on the SVN source code management tool, please see the SVN website and the online SVN book.
